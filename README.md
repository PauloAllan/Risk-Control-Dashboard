# Controle de Escoltas

Sistema web local em Python para cadastrar e acompanhar operações de escolta.

## Como executar

1. Instale as dependências: `python -m pip install -r requirements.txt`
2. Inicie o sistema: `streamlit run app.py`
3. Acesse o endereço exibido no terminal (normalmente `http://localhost:8501`).

Os dados ficam armazenados no arquivo local `painel_escoltas.db`, criado automaticamente na primeira execução.
